# Tastiest Backend - NestJS

## Starting with PM2
```bash
pm2 start yarn --interpreter bash --name tastiest-backend -- start
```

## Serving Documentation
```bash
yarn documentation:serve
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
